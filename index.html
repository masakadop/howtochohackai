スマホだけでできる！
GitHub Pages × Copilot 入門

このスライドは「最初の公開確認」と「Copilotでテトリス作成」の2ステップを、 小学生でも進められるように作っています。

目標1 index.html に Hello World を表示して公開確認する

目標2 index.html だけで Copilot にテトリスを作ってもらう

準備するもの（スマホ）

  • GitHubアカウント（無料）
  • スマホのブラウザ（Chrome / Safari）
  • インターネット
大事： GitHubアプリよりも、ブラウザで github.com を開く方が作業しやすいです。
画面右上の「⋯」メニューで
「PC版サイトを表示」をONにすると編集が楽です。

手順1：Hello Worldで公開確認

  1. GitHubで新しいリポジトリを作る（例：mypages）
  2. Add file → Create new file を押す
  3. ファイル名を index.html にする
  4. 中身に次の1行だけ書く：Hello World
  5. Commit changes を押す

手順1：GitHub PagesをONにする

  1. Settings タブを開く
  2. Pages を開く
  3. Build and deployment の Source を Deploy from a branch にする
  4. Branch を main / /(root) で保存
  5. 表示されたURLを開いて Hello World が出たら成功！
まずはこの確認が最重要。ここが通れば、次のテトリス作成も必ず進められます。

手順2：なぜCopilotを使うの？

理由はシンプルです。1つずつ難しいコードを覚えなくても、作りながら学べるからです。

  • 「こういうゲームを作りたい」を文章で伝えられる
  • 修正も「ここを直して」と会話で進められる
  • 小学生でも「作れた！」体験に早くたどり着ける
Copilotのコードはそのまま貼るだけでなく、必ず動作確認して1行ずつ理解していきましょう。

手順2：index.html だけでテトリス作成

  1. 同じ index.html を開いて編集
  2. Copilot Chat に次のお願いを送る：
    「index.html 1ファイルだけで動くテトリスを作って。
    HTML/CSS/JavaScriptを1つのファイルにまとめて、 スマホのタッチ操作（左右移動・回転・高速落下）ボタンも付けて。」
  3. 提案コードを index.html に貼る
  4. Commit changes を押す

テトリスをスマホ向けに完成させるコツ

  • 文字が小さすぎたら「フォントを大きくして」とCopilotに依頼
  • 操作しづらければ「ボタンを画面下に大きく配置して」と依頼
  • ゲームオーバー後に「再スタートボタンを追加して」と依頼
  • 色を変えたければ「ブロック色をカラフルに」と依頼
うまくいかないときは「エラーの原因を説明して、修正版を全部表示して」と送るのがコツ。

最終チェックリスト

  1. Hello World の公開URLにアクセスできる
  2. index.html をテトリス版に更新して再公開できる
  3. スマホでテトリスが操作できる
  4. 「なぜこのコードか」を1つ説明できる

ここまでできたら、あなたはもう「AIと一緒にアプリを作れる人」です！
